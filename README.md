# Personal-Budget-Calculator
A Python application that trackes budget based on income and expenses
## Overview
This project demnstrates basic Pyhon programming concepts applied to a budgeting scenario to track income and expenses.

## Features
- Tracks income and expenses
- calculates total spending
- Displays remaining balance
- Provides budget states as within or over budget

## Technologies used
- Python
- Microsoft Visual studio

## Design Notes
The application was designed to simulate financial tracking using basic programming logic, including loops, conditions and data structure.
